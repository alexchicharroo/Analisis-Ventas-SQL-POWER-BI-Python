# DASHBOARD DE VENTAS Y FACTURACION

## OBJETIVO DEL PROYECTO

El objetivo de este proyecto es analizar las ventas de la empresa y estudiar como se comportan según el lugar, el tipo de cliente, las fechas... De esta forma en el futuro se podrá optimizar la venta.



## HERRAMIENTAS

Para realizar el proyecto se ha utilizado POWER BI DESKTOP para realizar el diseño y la estructura del informe. Además, se ha utilizado POWER QUERY para modificar algunos datos como los números decimales.



Dentro de POWER BI, se ha utilizado herramientas como DAX para crear nuevas medidas como el ticket medio, o nuevas columnas como el importe. También se ha usado el modelado de datos para conectar las tablas unas con otras para poder aplicar algunos filtros.



## DESARROLLO

Primero se ha cambiado el formato de los números decimales para que no hubiese errores. Tras ello se ha utilizado el modelado para poder aplicar filtros y de esta forma poder aislar información según el interés y por último se ha creado la visualización de los datos para que se pueda ver de forma directa los resultados.

## CONCLUSIONES

Los refrescos y ensaladas son los productos que mayor ingresos dan a la empresa, ya que superan los 20 mil euros, en cambio, la pizza es de los productos que menos ingresos dan. 

En cuanto a las ciudades, en todas se suele vender casi lo mismo, los datos son muy cercanos, pero en las ciudades de origen del cliente si varía, siendo Zaragoza la que mayor dinero deja.

Para poder predecir, se ve que el análisis en el tiempo, es muy cíclico, con fechas de aumentos y fechas de picos bajos, por lo que se puede predecir y planificar según la fecha en la que se encuentre.
