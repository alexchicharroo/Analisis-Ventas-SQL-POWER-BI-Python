#PROYECTO DE ANALISIS COMERCIAL Y PROYECCION FUTURA

Proyecto realizado con 5000 datos sintéticos que simulan datos históricos de un negocio. Se quiere realizar un analisis sobre los datos y una proyección a futuro.

#Herramientas utilizadas

SQL: extracción de los datos de las tablas ventas, productos, clientes con join, group by, having...

POWER BI: creacion dashboard interactivo y métricas KPI como facturación total, ticket medio, unidades totales...

PYTHON: limpieza de datos, creación columna importe y gráficos.

IA: entendimiento soluciones IA con 70% datos para entrenamiento y 30% para test, con un error absoluto medio de 5.03€
